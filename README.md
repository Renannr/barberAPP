### Este aplicativo é um exemplo de rotina de login e cadastro
   Foram utilizado:
   * Styled Components;
   * Svg-transformer;
   * Stack Navigation;
   * Bottom Tab Navigation;
   * AsyncStorage;
   * Requisição a API (usando fetch);
   * Manipulação de contexto (usando useContext).
