### Este aplicativo é um exemplo de rotina de login e cadastro de usuário
Foram utilizados:
   * Styled Components;
   * Svg-transformer;
   * Stack Navigation;
   * Bottom Tab Navigation (com tab custom);
   * AsyncStorage;
   * Requisição a API (usando fetch);
   * Manipulação de contexto (usando useContext).
